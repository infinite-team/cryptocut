Description:
CryptoCut is a easy-to-use and user-friendly cross-platform application to communicate securely without changing a lot of things around us. you can code your message as easy as a shortcut and decode it with another. the only thing you need is to sync your cipher codes to communicate securely through any safe and unsafe communication framework. CryptoCut let you to make a human-readable secure code. simple and easy to use for daily communications if you care about your privacy. another important fact about this application is it's build with qt based technology for settings UI page and all of widgets are native to your OS. Even you can set program service(daemon) as startup and open configs page with one click on tray icon.
This app wasn't possible without ditching classes (!!!) and some C Scores so please help us to make this project more meaningful
